# Duo website

This uses Jekyll: https://jekyllrb.com/

## To run locally
```sh
# install Ruby (using asdf)
asdf add plugin ruby
asdf install

# install bundler
gem install bundler

# install ruby deps
bundle install

# install npm deps
npm i

# run it!
bundle exec jekyll serve -l

# or view draft blogs as if they were published
bundle exec jekyll serve -l --drafts
```
